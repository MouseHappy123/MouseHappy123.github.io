常用命令及操作

2023-05-13 2 min read # work

mac

command+r 刷新
command+y 浏览器历史
command + shift + 4（+ctrl） 部分截图（并复制）

shift + command + 左右箭头 选择行文字
option + 左右 光标按单词跳转
上+command+p vscode打开搜索栏
command + t vnote切换编辑预览模式
control + '-' VsCode跳转回之前的位置
shift + option + 上下 复制上下一行/段
option + z 压缩字符串到屏幕大小

iterm

command + = 启动
command + 左右 移动光标
command + 删除 整行删除

git

git stash 保存当前工作进度
git stash pop 恢复最新的进度到工作区

vim

shift + 4 行尾
shift + g 最后一行
gg 第一行
0 行首
dG 删除全部
w b 单词首字母跳；de从头删除单词
e ge 单词尾字母跳；dw从尾删除单词

可视化

shift + v 行模式，选中后d快速删除行
ctrl+v 块模式，多行同一位置批量增加，选中后，shift + i ，再esc，批量删除 选中后，d

sed head tail

在Linux中，可以使用以下命令来查看文件的第n行：

sed -n 'n p' filename
其中，n为要查看的行数，filename为要查看的文件名。

例如，要查看文件test.txt的第5行，可以使用以下命令：
sed -n '5 p' test.txt

另外，还可以使用以下命令来查看指定范围内的行：

查看文件的前n行：head -n filename
查看文件的后n行：tail -n filename
查看文件的第n行到第m行：sed -n 'n,mp' filename
查看文件的第n行到末尾行：sed -n 'n,$p' filename
例如，要查看文件test.txt的前10行，可以使用以下命令：
head -n 10 test.txt
